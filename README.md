### Main Usage
```bash
git-nav [path to the git repo]
```

#### Shortcuts

| Shortcut | Description                                                                                                          |
| --------- | -------------------------------------------------------------------------------------------------------------------- |
| `Q`     | Quit the program                          |
| `Enter`     | Opens the File Inside Neovim                          |
| `Left Arrow`    | Open the folder |
| `Right Arrow`    | Go back on step in the folders tree |
