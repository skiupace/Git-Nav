*   Edit Files In Neovim

<h2>🛠️ Installation Steps</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/skiupace/Git-Nav.git
```

<p>2. Make sure rust is installed</p>

<p>3. Build the project</p>

```
cargo build --release
```

<p>4. Run the project</p>

```
cargo run
```
